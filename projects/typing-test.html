Typing Test

How it works:

Getting the words:

the program sends requests to a server asking for a variable amount of words. the server returns an array with the amount of words requested.

the words are then stored in an array.

Displaying each word:

to display each letter of each word, I created a span for each word. if the letter was correctly typed, it would be given a certain class, and the same would happen if the word was typed inccorectly.

when a new word needed to be displayed, i would delete each span and create a new one for each letter of the new word, and store it in an array.

Highscores: (not yet added)

times for each word amount are stored via localstorage.

the only downside of this would be if the user clears his/her cache they lose their highscores.
